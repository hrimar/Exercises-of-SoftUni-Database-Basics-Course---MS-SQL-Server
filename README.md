
# Exercises of SoftUni DB Basics Course - MS SQL Server

The course covers:

- MSSQL Server, T-SQL;
- creating tables and relations between them;
- queries and nested queries;
- built-in functions;
- user-defined functions;
- stored procedures;
- transactions;
- triggers;
